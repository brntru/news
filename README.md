# StudyPoint

An educational platform built with React, Vite, TypeScript, and Tailwind CSS.

## Pages

- **Home** — Video hero + featured courses + testimonials
- **Courses** — Searchable, filterable course grid
- **Pricing** — Free / Student / Pro plan comparison
- **Resources** — Downloadable study materials
- **About Us** — Platform overview and community
- **Preview** — Features showcase
- **Login / Sign Up** — Tabbed auth form
- **Dashboard** — Student progress, schedule, and course list

## Getting Started

```bash
# Install dependencies
npm install

# Start dev server (http://localhost:3000)
npm run dev

# Build for production
npm run build

# Preview production build
npm run preview
```

## Stack

- React 18 + TypeScript
- Vite 6
- Tailwind CSS v4
- Framer Motion (page transitions)
- Wouter (client-side routing)
- Radix UI + shadcn/ui components
- TanStack Query

## Video Hero

Place your hero video at `public/hero.mp4`. The homepage will autoplay it muted and looped.
